API (NestJS) — MUST IMPLEMENT
Base URL: /v1
1) GET /v1/collections/:slug
Query params:
  cursor, limit, sort, orientation, color, size, material, onSale
Response:
{
  items: ProductCard[],
  nextCursor: string | null,
  total: number,
  facets: {
    orientation: { value: string, count: number }[],
    color: { value: string, count: number }[],
    size: { value: string, count: number }[],
    material: { value: string, count: number }[],
    onSale: { value: "true"|"false", count: number }[]
  }
}
Sort options:
- popular (rating desc, ratingCount desc)
- newest (createdAt desc)
- price_asc (minVariantPrice asc)
- price_desc (minVariantPrice desc)

2) GET /v1/products/:slug
Return product details + images + variants:
{
  product: {...},
  images: [...],
  variants: [
    { id, material, size, priceCents, sku, inStock }
  ],
  priceRange: { minCents, maxCents }
}

3) CART (session-based, no real payment)
- GET /v1/cart
- POST /v1/cart/items { variantId, qty }
- PATCH /v1/cart/items/:id { qty }
- DELETE /v1/cart/items/:id
Cart should persist via cookie-based sessionId (signed/uuid).
